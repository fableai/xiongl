## 🤝 贡献指南

### 安装包管理器
```sh
npm install pnpm -g
```

### 安装项目依赖
```sh
pnpm install
```

### 启动项目
```sh
pnpm run dev
```

### 构建
```sh
  pnpm run build
```

## commit 规范
- `feat`	    新增功能 feature
- `fix`	      修复 bug
- `docs`	    修改文档
- `style`	    样式，代码格式 (不影响代码运行的改动)
- `refactor`	重构优化
- `perf`	    性能，体验优化
- `test`	    单元测试
- `chore`	    构建过程或辅助工具改动
- `revert`    回退
- `release`	  版本发布
